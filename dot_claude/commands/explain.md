# コード解説

コードを明確かつ教育的に説明します。

## 手順

コードを説明する際:

### 1. 概要
- このコードの目的は何か？
- どのような問題を解決するか？
- システムのどこに位置するか？

### 2. ステップバイステップの分解
- ロジックの流れを説明
- 重要な設計判断とパターンを説明
- 重要な変数と関数をハイライト

### 3. 技術的詳細
- 使用されているアルゴリズムとその計算量
- 適用されているデザインパターン
- 活用されている言語固有の機能

### 4. コンテキストと関連性
- 依存関係とインポート
- コードベースの他の部分との連携
- 副作用や外部との相互作用

## 出力形式

```
## 概要
[コードが何をするかの簡潔な説明]

## 動作の仕組み
[ステップバイステップの説明]

## 重要な概念
- [概念1]: [説明]
- [概念2]: [説明]

## 使用例
[該当する場合、このコードの使用方法を示す]

## 関連
[関連するファイル、関数、概念に言及]
```

## ガイドライン

- コードの複雑さに応じて深さを調整
- 有用な場合は比喩を使用
- 非自明な動作をハイライト
- 「何を」だけでなく「なぜ」を説明
- 読者が学習中であると想定
